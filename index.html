GAnarchy on ganarchy.github.io

This is GAnarchy on ganarchy.github.io. Currently tracking the following projects:

  • A Boneless Datastructure Language: A small domain-specific language inspired by regular expressions, designed for matching all sorts of config files and data structures.
  • Anarki: Community fork of the Arc programming language, a dialect of Lisp with cleanly designed core operators designed for exploratory programming, particularly building web apps.
  • Converter Microwave: The Converter Microwave is an open hardware project for converting any cavity magnetron microwave into an inverter microwave.
  • GAnarchy: A Project Page Generator written in Python, focused on giving forks of a project the same visibility as the original repo.
  • HexArt: A pixel art editor?
  • Rust hexchat-plugin: Rust crate that allows one to write hexchat plugins in Rust.
  • Tiny Desktop Portal: A modular and lightweight alternative to `xdg-desktop-portal`, primarily for use with *non*-flatpak applications.

Consider visiting https://www.reddit.com/r/GAnarchy/ to learn more and talk about GAnarchy.

If you'd like to see another project here, or want your repo tracked in one of these project pages, open an issue on github: https://github.com/ganarchy/ganarchy.github.io/issues.
If you'd like to see your project here, you also need to create a "project commit", see the GAnarchy README for details.
