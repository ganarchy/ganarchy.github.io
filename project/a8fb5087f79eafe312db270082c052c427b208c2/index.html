[Project] Rust Programming Language MMO RFC

This is an experiment to make a whole RFC using an Massively Multiplayer Online-like process. The idea being that such RFCs would get written faster, have less issues, while also dismantling systemic barriers which get in the way of contributing to the RFC process.
